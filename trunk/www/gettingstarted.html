Aerith: Getting Started

Requirements

To get Aerith running you must first download the source code to Aerith, the SwingX project from Swing Labs, and JOGL. You also need to be running a recent build of Mustang. You will also need to compile using NetBeans. Because Aerith is graphics intensive and uses cutting edge technology you need to run it on a computer with a 3d graphics accelerator and Open GL drivers installed.

If you have any problems compiling or running Aerith please post a question to the users mailing list.

Compliation and Installation

  1. Download and install the most recent Mustang beta (Java SE 6.0) from the Mustang project.
  2. Download JOGL (Java Open GL bindings) from the JOGL homepage. Be sure you get both the jogl.jar file and the native libraries for your platform. You can get precompiled binaries from the download page.
  3. Install JOGL by unjarring the jogl-natives[your platform].jar file. It should create several native libraries. Put the native libs in your library path and the jogl.jar in your classpath.
    • Mac OS X: Put the jogl.jar and the *.jnilib files in the /Library/Java/Extensions directory
    • Windows [need help on this one]
    • Unix [need help on this one]
  4. Check out the code for SwingX from swingx.dev.java.net. You will need CVS tools installed. (Built into NetBeans)
  5. Check out the code for Aerith. You will need Subversion tools installed. (Built into NetBeans 5.5)
  6. Open the Aerith project in NetBeans and resolve the broken references. You will need to tell the Aerith project where you have downloaded the SwingX project, where the JOGL jar is, and where your lib directory is.
  7. Compile and run Aerith. If it you have problems please post it to the mailing list.
